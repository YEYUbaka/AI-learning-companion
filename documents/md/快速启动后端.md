# 🚀 快速启动后端服务

## 📍 当前状态

✅ `.env.template` 文件已存在，包含你准备好的所有API密钥
✅ `.env` 文件已存在

## 🎯 启动步骤

### 方法1：使用启动脚本（最简单）

1. **打开PowerShell或命令提示符**
2. **进入backend目录**：
   ```powershell
   cd "F:\Cursor projects\Web\backend"
   ```
3. **运行启动脚本**：
   ```powershell
   .\start.bat
   ```

### 方法2：双击启动脚本

1. 在文件资源管理器中打开 `backend` 文件夹
2. 双击 `start.bat` 文件

## ✅ 启动成功的标志

启动成功后，你应该看到：

```
========================
启动服务...
========================

访问地址: http://127.0.0.1:8000
API文档: http://127.0.0.1:8000/docs

INFO:     Uvicorn running on http://127.0.0.1:8000
INFO:     Application startup complete.
✅ 数据库表创建成功
```

## 🔍 验证服务

启动后，在浏览器访问：

- **健康检查**：http://127.0.0.1:8000/health
- **API文档**：http://127.0.0.1:8000/docs
- **Providers接口**：http://127.0.0.1:8000/api/v1/ai/providers

## 📝 关于.env文件

### 如果.env文件配置不完整

`.env.template` 文件已经包含了你准备好的API密钥，你可以：

1. **查看模板文件**：
   ```powershell
   cd "F:\Cursor projects\Web\backend"
   notepad .env.template
   ```

2. **复制配置到.env**：
   - 打开 `.env` 文件
   - 从 `.env.template` 复制你需要的配置
   - 保存文件

3. **重启后端服务**（如果已启动）

## ⚠️ 重要提示

1. **启动脚本会自动**：
   - 检查Python环境
   - 激活虚拟环境
   - 安装依赖
   - 启动服务

2. **如果启动失败**：
   - 检查Python是否已安装
   - 检查端口8000是否被占用
   - 查看错误信息

3. **停止服务**：
   - 在PowerShell窗口按 `Ctrl+C`

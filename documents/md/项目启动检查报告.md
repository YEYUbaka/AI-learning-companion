## 🐛 常见问题

### 后端无法启动
1. **检查Python环境**:
   ```bash
   cd backend
   python --version
   ```

2. **检查虚拟环境**:
   ```bash
   cd backend
   .\venv\Scripts\activate
   pip list
   ```

3. **手动启动**:
   ```bash
   cd backend
   .\venv\Scripts\activate
   uvicorn main:app --reload --port 8000
   ```

### 前端无法访问
1. **检查端口占用**:
   ```bash
   netstat -ano | findstr :5173
   ```

2. **重新启动前端**:
   ```bash
   cd frontend
   npm run dev
   ```

### API调用失败
1. **检查.env配置**:
   - 确保 `backend/.env` 文件存在
   - 确保配置了 `DEEPSEEK_API_KEY` 或 `AI_API_KEY`
   - 确保 `AI_PROVIDER=deepseek`

2. **检查CORS设置**:
   - 后端已配置允许所有来源
   - 如果还有问题，检查浏览器控制台错误

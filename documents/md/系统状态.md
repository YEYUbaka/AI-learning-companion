## ⚠️ 已知问题

### 数据库连接失败
- **原因**：SQL Server 未启动或未安装
- **影响**：用户注册功能不可用
- **解决方案**：目前不影响AI问答功能测试
- **修复**：安装并启动 SQL Server 或使用 SQLite

### 前端可能需要稍等片刻
- **原因**：Vite开发服务器正在启动
- **解决**：等待5-10秒后刷新页面

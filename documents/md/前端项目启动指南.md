# 🚀 前端项目启动指南

## ⚠️ 前提条件

**必须安装 Node.js 才能运行前端项目！**

## 📦 第 1 步：安装 Node.js

### 方法一：手动下载安装（推荐）

1. **下载 Node.js**
   - 访问：https://nodejs.org/zh-cn/
   - 下载 **LTS 版本**（推荐 v18.x 或 v20.x）
   - 点击 Windows 安装包（.msi）

2. **安装 Node.js**
   - 运行下载的安装程序
   - ✅ **重要**：勾选 "Add to PATH" 选项
   - 点击"安装"
   - 完成安装

3. **验证安装**
   打开新的 PowerShell，运行：
   ```powershell
   node --version
   npm --version
   ```
   
   如果显示版本号，说明安装成功！

### 方法二：使用快捷脚本

我已经为你创建了检查脚本：
```
双击运行：检查Node.js.bat
```

如果没有安装，运行：
```
双击运行：安装Node.js.bat
```
这会打开浏览器到下载页面。

## 🎯 第 2 步：启动前端项目

### 方法一：命令行启动

```powershell
# 1. 进入前端目录
cd "F:\Cursor projects\Web\frontend"

# 2. 安装依赖（首次运行）
npm install

# 3. 启动开发服务器
npm run dev
```

### 方法二：使用快捷脚本

我还可以为你创建一个批处理脚本，双击即可启动：
```batch
前端启动.bat
```

## ⚙️ 完整流程总结

1. **安装 Node.js**
   - 下载：https://nodejs.org/
   - 安装：勾选 "Add to PATH"
   - 验证：`node --version`

2. **安装前端依赖**
   ```bash
   cd frontend
   npm install
   ```

3. **启动前端服务**
   ```bash
   npm run dev
   ```

4. **确保后端已启动**
   - 后端地址：http://127.0.0.1:8000
   - 运行 `backend/start.bat`

5. **测试完整功能**
   - 注册用户
   - 登录系统
   - 使用 AI 问答

## 🐛 常见问题

### Q: npm 命令未找到？
A: Node.js 未安装或未添加到 PATH。重新安装并勾选"Add to PATH"。

### Q: 端口 5173 被占用？
A: 修改 `frontend/vite.config.js` 中的 port 配置。

### Q: 模块导入错误？
A: 运行 `npm install` 重新安装依赖。

### Q: 无法连接后端？
A: 确保后端已启动（运行 `backend/start.bat`）

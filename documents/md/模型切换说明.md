# 🔄 模型切换说明

## 📍 当前切换方式（方法1：配置文件）

### 位置
在 `backend/.env` 文件中修改 `AI_PROVIDER` 配置

### 步骤
1. 打开 `backend/.env` 文件
2. 修改 `AI_PROVIDER` 的值：
   ```ini
   # 支持的选项：
   AI_PROVIDER=deepseek    # DeepSeek（默认）
   AI_PROVIDER=wenxin       # 文心一言
   AI_PROVIDER=xinghuo      # 星火
   AI_PROVIDER=chatglm      # ChatGLM
   AI_PROVIDER=moonshot     # Moonshot
   ```
3. 确保配置了对应模型的API密钥
4. **重启后端服务**（必须！）

### 缺点
- 需要重启后端服务
- 不够直观
- 需要编辑配置文件
